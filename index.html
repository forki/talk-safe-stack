S for Suave

https://suave.io

  • Standalone, lightweight & composable web server
  • Alternatives:
    • Giraffe (on top of ASP.NET Core), or Saturn
    • Freya (web-machine like)

A for Azure

https://azure.microsoft.com

  • Cloud provider from Microsoft
  • Alternatives:
    • Amazon Web Services
    • Google Cloud Platform

F for Fable

http://fable.io

  • F# to JS compiler powered by Babel
  • Alternatives:
    • WebSharper

E for Elmish

https://fable-elmish.github.io/elmish

  • UI library, inspired by Elm
  • Alternatives:
    • WebSharper models

Creating SAFE project

Prerequisites:

  • dotnet SDK 2
  • node.js
  • yarn or npm
  • .NET Framework or mono for build process*

* removing this dependency is WIP

Creating SAFE project

  • Install SAFE template: dotnet new -i SAFE.Template
  • Create project from template: dotnet new SAFE
  • Build & run: build run
  • Wait for build to finish: app opens up in browser

Architecture

  • Shared code
    • Fable.Remoting
  • Server side
    • Suave
  • Client side
    • Fable
    • Elmish
    • React under the hood

Talking to server side

  • Server refresh (dotnet watch)
  • Triggering calls from Client side
